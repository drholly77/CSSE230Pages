Method for Chip & Conquer

Let:

  • Branching factor: b = 1
  • Chipping factor: c > 0
  • f(n) be a function

  • Recurrence: T(n) = bT(n - c) + f(n)

Then T(n) can be bounded as follows:

  • If f(n) is a polynomial: nα, then T(n) ∈ Θ(nα+1)
  • If f(n) is of the form: log2(n), then T(n) ∈ Θ(nlog2(n))
